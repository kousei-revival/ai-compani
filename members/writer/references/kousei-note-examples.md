# Examples

## Example 1: 運動習慣は、意志力じゃない。仕組みで決まる。

### Input

```text
みかみ寄せで、こうせいの運動習慣noteを最初から1本書いて
```

### Output Strategy

- 文体リズムは `みかみ寄せ`
- でも題材は `こうせい × 運動習慣`
- 病棟で見てきたリアルから始める
- 中盤で `意志力モデル` を崩す
- 後半で `小さな設計` に落とす
- 最後は `あなたのせいじゃない` に着地する

### Result Snapshot

```md
# 運動習慣は、意志力じゃない。仕組みで決まる。

どうも！
理学療法士のこうせいです。

回復期リハビリテーション病棟で、
たくさんの患者さんと関わってきました。

そこで何度も見てきた光景があります。

入院中は、
運動できるんです。

リハビリにも来る。
自主トレもする。
「退院してからも頑張ります」って言う。

でも。

退院して、
数週間とか、
数ヶ月とか経つと、
やめてしまう人が少なくない。

なぜか。

$ 運動習慣は、意志力で決まるものじゃないからです。

仕組みで決まるんです。
```

### Why This Works

- 冒頭で病棟のリアルを置いて一気に引き込んでいる
- `でも。` `なぜか。` `$ 結論` でみかみ的なリズムを移植している
- みかみの固有文脈ではなく、こうせいの専門性に完全に置き換えている
- 読者を責めずに、構造の問題へ視点を移している

### Required Elements Present

- 名乗りが `理学療法士のこうせい`
- 一人称が `ぼく`
- 病棟の具体エピソードがある
- 結論が `仕組み` に着地している
- `P.S. / P.P.S. / P.P.P.S.` まで含む note 完成形
