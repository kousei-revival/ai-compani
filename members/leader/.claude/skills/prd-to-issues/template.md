- Parent PRD: [参照先]

  - [ ] 条件2
- Blockers:

  - 何を作るか

  - [ ] 条件1

## メモ

- 各 issue は単独で進捗確認できる形にする
- まずは `AFK` を増やせないか考える
- 依存が複雑なら、先に `prd-to-plan` で整理してから分解する
