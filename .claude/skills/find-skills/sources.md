# Skill Sources

## まず見る場所

### 1. Anthropic公式

- 入口: `github.com/anthropics/skills`
- まず確認するもの:
  - Meta Skills
  - Office / Frontend / Review 系
- 判断ポイント:
  - 汎用性が高いか
  - 手順が再利用しやすいか

### 2. Matt Pocock

- 入口: `github.com/mattpocock/skills`
- まず確認するもの:
  - `write-a-skill`
  - `grill-me`
  - `tdd`
  - `write-a-prd`
- 判断ポイント:
  - 実務フローに直結しているか
  - 設計や開発でそのまま回せるか

### 3. SkillsMP

- 入口: `skillsmp.com`
- 役割:
  - 既存Skillの有無をざっと確認する検索起点
- 注意:
  - 取得がブロックされることがある
  - 見つけたらGitHub本体や配布元も確認する

## 探すときのメモ

- まず `skill名` より `やりたい作業` で探す
- 1件見つかったら、その作者の他Skillも見る
- 良さそうでも、説明文が弱いなら発火しにくい
- 自分用化では `内容` だけでなく `description` も必ず直す
