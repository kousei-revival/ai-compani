# brand-guidelines

見た目の一貫性を、後付けで整える。

## 基本ルール

- 先にブランドの核を確認する
- 色だけでなく、書体、余白、言葉の強さも揃える
- 対象物に無理やり当てない

## 進め方

### Step 1: ブランド要素を整理する

- 主色
- 補助色
- 書体
- トーン
- 避けたい表現

### Step 2: 対象に反映する

- UI
- スライド
- 文書

### Step 3: 崩れを確認する

- 色が散っていないか
- 書体ルールが崩れていないか
- ブランドらしさが残っているか

## 出力フォーマット

```md
## Brand Core
- 色
- 書体
- トーン

## Applied Rules
- rule1
- rule2
```
